# POWER_BI
[Content_Types].xml: This is a standard package configuration file specifying the content formats contained inside the archive.DataModelSchema: This contains the core back-end data architecture. It defines your tables, relationships, data connections, calculated columns, and DAX measures used for tracking COVID statistics.DiagramLayout: A system file that stores the visual model layout of tables and their connections in Power BI's "Model View."Report/Layout: This critical file holds the front-end layout configuration. It defines the visual canvas, including where your dashboard charts, filters, metrics, and text boxes sit on the screen.
